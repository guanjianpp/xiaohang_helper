# 小航 · 郑州航院校园信息AI助手
## 项目简介
本项目是基于Streamlit开发的校园智能问答网页应用，面向郑州航空工业管理学院新生、在校生、教师三类人群，通过读取本地Markdown校园知识库，调用硅基流动大模型API实现校园信息智能答疑。项目设置严格提示词规则，限制AI仅依靠本地资料作答，避免编造流程、电话、费用等虚假信息，覆盖入学报到、办事流程、交通出行、反诈应急等校园常用场景。

## 使用技术
1. Streamlit：网页可视化界面搭建
2. requests：调用大模型API接口
3. Python标准库：time、pathlib
4. 版本控制：Git & GitHub

## 实现全部功能
1. 用户身份切换：新生/在校生/教师，差异化AI回答风格
2. 分类标签快捷提问：新生指南、办事流程、应急防骗三类按钮
3. 多轮对话记忆，支持上下文追问
4. 问答历史存储展示，支持一键清空历史记录
5. API加载动画，请求时显示「小航正在思考...」
6. 回答底部小字展示回答字数、接口耗时
7. 输入长度限制，超长提问提前拦截不调用接口
8. 完善异常捕获：超时、断网、密钥失效、数据缺失提示
9. AI防幻觉强制规则，自动标注信息来源，无资料给出学校总机兜底
10. 底部静态校内部门电话黄页，方便紧急查询
